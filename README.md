# Live Outputs

This folder contains outputs generated during flood events.

Files generated here:
- `priority_report_YYYY-MM-DD.csv` — ranked union priority table
- `netrokona_dashboard_YYYY-MM-DD.html` — interactive Folium map

These files are not committed to the repository as they are
generated fresh each time the pipeline runs.

Run `notebooks/03_flood_impact_demo.ipynb` to generate outputs
for the August 2024 demo scenario.
